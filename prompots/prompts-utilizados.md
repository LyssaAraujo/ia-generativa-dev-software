**## 📌 Prompt 3 — Geração de Código
### Prompt utilizado**
Crie um código Python usando IA
Problema encontrado:
O código ficou incompleto
Solução aplicada:
Solicitei explicações linha por linha
Prompt melhorado:
Crie um exemplo completo em Python com comentários explicando cada parte do código
Resultado:
O código ficou mais didático e funcional
